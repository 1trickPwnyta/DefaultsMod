Default storyteller settings
Set the default storyteller/difficulty settings to use for a new colony, including custom difficulty settings.
Default world settings
Set the default world, map, and faction settings to use for a new colony.
Default colonist schedules
Set custom default schedules for colonists. You can create alternating schedules and the mod will cycle between them when assigning them to new colonists.
Default policies
Set custom default apparel, food, drug, and reading policies that will be copied to each new colony.
Includes additional settings to automatically assign certain policies to certain types of pawns. For example, you can ensure new children to the colony get the no-drug policy.
Default baby feeding settings
Set the default autofeed setting for colonists based on relationship to baby and whether they are lactating. Also allows you to set which foods babies can be fed by default.
Biotech only.
Default medicine settings
Set the default medicine settings to use for a new colony.
Includes additional settings for default medicine type and amount to carry and whether guests should carry your medicine as well.
Default reward preferences
Set the default reward preferences to use for a new colony.
Default expanded resource categories
Set which resource tree categories are expanded by default in categorized mode.
Default storage settings
Configure the default settings for stockpile zones, dumping stockpile zones, and storage buildings such as shelves. You can also create your own custom stockpile zone types with their own settings.
Includes an additional setting for allowing removing items from outfit stands by default (Odyssey only).
Default workbench bills
Set default bills for workbenches that will populate automatically upon building a workbench.
Includes additional settings for default ingredient search radius, skill range, and store mode. You can remove the limit of 15 bills per workbench as well.
Default allowed areas
Create default allowed areas for new colonies and control which types of pawns get automatically assigned to which areas.
Includes additional settings for whether parent allowed areas are still inherited by their children.
Default fishing zone settings
Configure the default settings for fishing zones.
Odyssey only.
Default work priorities
Create default work priorities for new colonists based on their passions and skills.
Includes an additional setting for using manual work priorities by default.
Default play settings
Configure default play settings. These are generally the toggle buttons in the bottom-right corner of the screen.
Miscellaneous defaults
Other default settings that don't fit neatly into one of the other categories. These include:
  • Starting xenotype: Set the default xenotype for colonists in the character creation screen. (Biotech only.)
  • Hostility response: Set the default hostility response to use for new colonists.
  • Pregnancy approach: Set the default pregnancy approach that new romantic colonist relationships will take. (Biotech only.)
  • Plant type: Set the default plant type for new growing zones.
  • Auto-select caravan supplies: Choose whether this option will be enabled when opening a dialog to form a caravan.
  • Temperature targets: Set the default temperature targets for heaters and coolers.
  • Mech work modes: Set the default work modes for the initial two mechanitor control groups as well as a mode for all additional control groups granted by control sublinks. (Biotech only.)
